==============================================================
YOUTUBE VIEWER PRO v2.0
==============================================================

BASEADO NO YtbView5.1 - VERSAO MELHORADA E GRATUITA

==============================================================
COMO INSTALAR
==============================================================

1. Instale o AutoHotkey
   - Baixe em: https://www.autohotkey.com/
   - Instale a versao mais recente

2. Extraia o arquivo ZIP em uma pasta
   - Recomendado: C:\YouTubeViewerPRO\

3. Execute o script
   - Clique duas vezes em: YouTube_Viewer_PRO_v2.ahk

==============================================================
ARQUIVOS INCLUSOS
==============================================================

- YouTube_Viewer_PRO_v2.ahk  : Script principal
- proxy_v2.txt               : Lista com 100 proxies
- README.txt                 : Este arquivo

==============================================================
COMO USAR
==============================================================

1. ABRA O SCRIPT
   Clique duas vezes em YouTube_Viewer_PRO_v2.ahk

2. CARREGUE OS PROXIES
   Clique em "CARREGAR PROXIES" e selecione o arquivo proxy_v2.txt
   (O script ja carrega automaticamente se o arquivo estiver na mesma pasta)

3. COLE O LINK DO VIDEO
   Insira o link do YouTube no campo indicado

4. CONFIGURE:
   - Quantidade de visualizacoes
   - Tempo minimo e maximo (segundos)
   - Trocar IP a cada X videos

5. CLIQUE EM INICIAR
   O Microsoft Edge abrira automaticamente

6. USE O PC NORMALMENTE!
   O script roda no Edge, voce pode usar o Chrome para trabalho

==============================================================
FORMATO DO PROXY
==============================================================

IP:PORTA:USUARIO:SENHA

Exemplo:
194.38.19.174:6736:nypkwabo:b9l2ztpk81vl

Voce pode adicionar mais proxies ao arquivo proxy_v2.txt
seguindo este formato (um por linha).

==============================================================
FUNCIONALIDADES

- Interface grafica completa
- Log de atividades em tempo real
- Contador de videos e tempo
- Pausar/Retomar a qualquer momento
- Roda no Edge (voce usa o Chrome)
- Comportamento humanizado
- Sem necessidade de licenca

==============================================================
ATALHOS DE TECLADO
==============================================================

F12 = Pausar/Retomar script
ESC = Encerrar script

==============================================================
DICAS
==============================================================

1. USE TEMPOS VARIADOS
   Configure tempo minimo 45s e maximo 120s para parecer
   mais natural

2. TROQUE IP REGULARMENTE
   Configure para trocar IP a cada 3-5 videos

3. NAO ABUSE
   Nao coloque quantidades muito altas de uma vez

4. VERIFIQUE OS PROXIES
   Clique em "TESTAR PROXY" para verificar se esta funcionando

==============================================================
ERROS COMUNS
==============================================================

ERRO: "Proxies nao carregados"
SOLUCAO: Clique em CARREGAR PROXIES e selecione o arquivo

ERRO: "Edge nao abriu"
SOLUCAO: Verifique se o Edge esta instalado

ERRO: Proxy nao funciona
SOLUCAO: Alguns proxies podem estar offline, o script
tentara automaticamente o proximo
